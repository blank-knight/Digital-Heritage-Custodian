# 数字遗产托管员 (Digital Heritage Custodian)

一个基于区块链的数字资产继承系统，让用户安全、自动地将数字资产转移给指定继承人。

## 功能特性

- **链上心跳检测**：监控用户90天无活动自动触发继承
- **多重签名验证**：白名单验证者确认继承流程
- **资产自动转移**：将ERC-20代币转移给继承人
- **Monad网络支持**：部署到Monad测试网络

## 项目结构

```
├── contracts/          # 智能合约
├── frontend/           # React前端应用
├── docs/              # 文档
└── scripts/           # 部署脚本
```

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发环境

```bash
npm run dev
```

### 3. 部署到Monad测试网

```bash
npm run deploy
```

## 技术栈

- **智能合约**：Solidity + Hardhat
- **前端**：React + Web3.js + Tailwind CSS
- **区块链**：Monad测试网
- **钱包**：MetaMask
