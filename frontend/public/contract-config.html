设置合约地址

请输入已部署的合约地址，设置完成后刷新应用页面即可生效。

配置指南

步骤1：部署合约

# 进入contracts目录
cd contracts

# 安装依赖
npm install

# 复制并编辑环境变量文件
cp env.example .env
# 编辑.env文件，设置PRIVATE_KEY和MONAD_RPC_URL

# 部署合约到Monad测试网
npm run deploy

步骤2：获取合约地址

部署完成后，合约地址会显示在控制台输出中，并保存到deployment.json文件中。

步骤3：配置前端

使用本页面输入合约地址并保存，或直接编辑frontend/src/config/contractConfig.js文件。
